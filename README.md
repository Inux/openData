# openData
openData

## Start webserver
* install go [https://golang.org/dl/](https://golang.org/dl/)
* open terminal and navigate to project
* go run server.go 
* open [http://localhost:3000](http://localhost:3000)

## Add more information to Nodes & Links Tooltip

* check method getNodeText() in graph.js for Nodes
* check method getLinkText() in graph.js for Links
